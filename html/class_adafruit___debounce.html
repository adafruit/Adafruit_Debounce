◆ begin() [1/2]

void Adafruit_Debounce::begin ( )

Initialize the debounce object.

This function sets the pin mode based on the polarity and initializes the button states. This assumes that the pin is GPIO (>= 0), otherwise please use the other begin(bool) which allows you to initialize the button start state.
